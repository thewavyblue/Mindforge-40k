What is Mindforge 40k?

In Mindforge 40k, you are testing yourself against your selected army's stats in order to develop a faster memory recall when it matters most...at the table.

Game setup

To setup each game, select an army and a category. These are combined into a question set that you are then tested against.

The game will begin immediately when the start button is clicked/tapped.

To answer a question, click/tap inside the left # input field and click/tap the submit button.

The answer appears in the field to the right of the submit button.

If your answer matches the answer input field flashes green: well done, you answered a question correctly!

If your answer doesn't match the answer input field flashes red: hard luck, try again on the next question.

End game

Currently, only timed mode has an 'end game' screen where you can view the results of your game (High scores leaderboard coming soon).

And that is all you need to know to play Mindforge 40k!

Are you ready to test yourself against your favourite army? Return to the home screen to select a game mode.
